Sobre o Projeto

O Mapeia ONG é uma solução tecnológica desenvolvida para conectar doadores às ONGs próximas às suas localidades, facilitando o processo de doação e promovendo ações sociais mais eficazes. A plataforma centraliza informações importantes como localização, horário de funcionamento, tipos de doação aceitos e formas de contato, tornando o ato de doar mais acessível e direto.

Nosso objetivo é empoderar tanto as ONGs quanto os doadores, criando uma rede de apoio onde todos possam contribuir para um impacto positivo na sociedade.
Problema que Resolvemos

Embora existam aplicativos que facilitam doações financeiras, poucos oferecem suporte direto para doações físicas ou conexão geolocalizada com ONGs. O Mapeia ONG resolve essa lacuna, permitindo que as pessoas identifiquem as necessidades específicas de instituições próximas e saibam exatamente como e onde doar.

Com isso, superamos barreiras como falta de informação, dificuldade de acesso e baixa transparência no processo de doação.
Principais Funcionalidades

    Localização de ONGs: Encontre ONGs próximas usando mapas interativos e filtros personalizados.
    Busca por Necessidades: Descubra ONGs que precisam de doações específicas, como alimentos, roupas ou medicamentos.
    Cadastro de ONGs: Organizações podem se registrar na plataforma, detalhando suas campanhas e necessidades.
    Feedback de Doações: Permita que ONGs enviem relatórios sobre o impacto gerado pelas doações recebidas.
    Interface Amigável: Design intuitivo e acessível, projetado para facilitar a navegação e a interação.

Diferenciais do Mapeia ONG

    Geolocalização Avançada: Integração com mapas e dados de localização precisos, fornecidos pela API Nominatim e Leaflet.
    Transparência e Conexão: O aplicativo cria um canal direto entre doadores e ONGs, promovendo confiança e engajamento.
    Inclusão e Acessibilidade: Aberto a qualquer ONG, independente de tamanho ou área de atuação, e pensado para ser acessível a diferentes públicos.

Arquitetura e Tecnologias Utilizadas

    Frontend:
        Vue.js para uma interface dinâmica e responsiva.
        Tailwind CSS para uma estilização moderna e consistente.

    Backend:
        C# e .NET Framework, oferecendo robustez e escalabilidade.
        EntityFramework para simplificação do acesso a dados.

    Banco de Dados:
        MySQL para armazenamento seguro e eficiente de informações.

    Mapeamento e Localização:
        API Nominatim e Leaflet para integração de mapas interativos.

    Design e Prototipagem:
        Figma para a criação de interfaces e protótipos funcionais.

Impacto Esperado

Com o Mapeia ONG, buscamos aumentar significativamente o número de doações realizadas de maneira local, proporcionando maior visibilidade para ONGs que, muitas vezes, enfrentam dificuldades para alcançar doadores. Acreditamos que o aplicativo pode servir como uma ponte entre a solidariedade e a necessidade, gerando benefícios reais para a sociedade como um todo.
